Getting Started with Vecore

Build and train neural networks with raw C++ and CUDA — no Python required.

What is Vecore?

Vecore is a GPU-accelerated deep learning framework built entirely from scratch in C++ and CUDA. It implements:

  • A multidimensional Tensor with automatic differentiation (autograd)
  • Neural network layers (Dense/Linear)
  • Loss functions (MSE, Cross Entropy)
  • Optimizers (SGD)
  • GPU acceleration via NVIDIA cuBLAS with TF32 Tensor Core support
  • A high-level Sequential API inspired by PyTorch
Info: Vecore achieved 96.23% test accuracy on MNIST — outperforming PyTorch's 95.61% on the same architecture and hyperparameters.

Prerequisites

  • C++17 compatible compiler (GCC 9+ or Clang 10+)
  • NVIDIA GPU with CUDA support (Compute Capability 7.0+)
  • CUDA Toolkit 11.0+
  • CMake 3.18+
  • OpenBLAS

1. Preparing the Data

Deep learning begins with data. In Vecore, whether you're working with images, text embeddings, or audio waveforms, everything is represented as a Tensor. For this tutorial, let's create a synthetic dataset to train a model on a classic non-linear problem: the XOR gate.

// 1. Preparing the Data (XOR Problem)
// X has shape {4, 2} (4 examples, 2 features)
auto X = vc::Tensor<float>({4, 2});
X.set_data({0, 0,   0, 1,   1, 0,   1, 1});

// y has shape {4, 1} (4 labels)
auto y = vc::Tensor<float>({4, 1});
y.set_data({0,      1,      1,      0});

// Move data to GPU for blazing fast training
X = X.to("cuda");
y = y.to("cuda");

2. The Manual Way: Building Layers

Before introducing abstractions, let's see how a neural network operates under the hood. A neural network is just a sequence of linear transformations and non-linear activations. Let's create two dense (fully-connected) layers manually.

// Layer 1: 2 inputs -> 8 hidden neurons, ReLU activation
vc::nn::Dense<float> layer1(2, 8, "relu");

// Layer 2: 8 hidden neurons -> 1 output neuron, Sigmoid for binary classification
vc::nn::Dense<float> layer2(8, 1, "sigmoid");

// Move parameters to the same device as our data
layer1.to("cuda");
layer2.to("cuda");

3. Forward Pass & Backpropagation

Vecore's autograd engine automatically tracks operations. When you pass X through your layers, Vecore builds a computational graph in the background. When you compute the loss and call .backward(), Vecore traverses the graph backward, automatically calculating the gradients for every weight.

// Forward Pass: Pass input through layers
auto hidden = layer1.forward(X);
auto out = layer2.forward(hidden);

// Compute Mean Squared Error Loss
auto diff = out - y;
auto loss = (diff * diff).mean();

// Backpropagate to compute gradients
loss.backward();

std::cout << "Loss: " << loss.item() << std::endl;

4. Abstracting it away: The Sequential API

Doing this manually for a massive 50-layer architecture like ResNet would be a nightmare. This is why Vecore introduces the Sequential model and Optimizers. They abstract away the tedious parameter management and weight updates.

Here is the exact same training loop, but using the idiomatic, PyTorch-like API:

#include "vecore/nn.h"
#include "vecore/optim.h"
#include "vecore/sequential.h"

int main() {
    // 1. Data Preparation
    auto X = vc::Tensor<float>({4, 2}).to("cuda");
    auto y = vc::Tensor<float>({4, 1}).to("cuda");
    X.set_data({0,0, 0,1, 1,0, 1,1});
    y.set_data({0,   1,   1,   0});

    // 2. Build the Model cleanly
    Sequential model({
        vc::nn::Dense<float>(2, 8, "relu"),
        vc::nn::Dense<float>(8, 1, "sigmoid")
    });
    model.to("cuda");
    
    // 3. Create an Optimizer
    vc::optim::SGD<float> optimizer(model.parameters(), 0.1);
    
    // 4. The Training Loop
    for(int epoch = 0; epoch < 1000; epoch++) {
        optimizer.zero_grad();               // Clear old gradients
        auto out = model.forward(X);         // Forward pass
        auto loss = vc::nn::mse_loss(out, y);// Compute loss
        loss.backward();                     // Backpropagate
        optimizer.step();                    // Update weights
    }
    
    // 5. Inference (Evaluation)
    auto predictions = model.forward(X);
    std::cout << "Predictions: " << predictions.to_string() << std::endl;
    
    return 0;
}
