Data Loading

struct Sample — The data unit

  • image (Tensor)
  • target (Tensor)
  • filepath (string)
  • label (int)

vc::data::Dataset — Abstract base class

Pure virtual methods:

  • size_t len()
  • Sample get_item(size_t index)

vc::data::InMemoryDataset — RAM-resident dataset

Designed for subclassing.

  • Holds std::vector<Sample> data
  • Methods: len(), get_item(), get_raw_data()
