Sequential — The Model Container

Overview: High-level container that chains Dense layers and provides a complete training pipeline with async GPU execution.

Constructor

Sequential(std::initializer_list<vc::nn::Dense<float>> layers)

Example:

Sequential model({ Dense(784, 2048, "relu"), Dense(2048, 10, "none") });

Methods

  • Sequential& to(const std::string& device) — transfers all layers
  • Tensor<float> operator()(const Tensor<float>& x) — sequential forward pass
  • vc::vector<Tensor<float>*> parameters() — all parameters across all layers
  • void save(const std::string& filepath) — exports weights as JSON

The Main Training Method

void fit(InMemoryDataset& dataset, int epochs, float lr, int batch_size)

This method implements an async GPU pipeline:

  1. Pre-packs dataset into contiguous pinned memory
  2. Creates CUDA streams for double-buffered transfers
  3. Overlaps data transfer with GPU compute
  4. Uses fused cross-entropy kernel on GPU
  5. Synchronizes once per epoch

Training Pipeline Diagram

[Pinned Memory]       [CUDA Stream 1]       [CUDA Stream 2]
    Batch N   ---H2D---> Transfer N 
                            |
    Batch N+1            Compute N   ---H2D---> Transfer N+1
                                                   |
    Batch N+2                                   Compute N+1

Full Training Example

// Load dataset
MNISTDataset train_dataset("data/train-images.idx3-ubyte", "data/train-labels.idx1-ubyte");

// Define model
Sequential model({
    vc::nn::Dense<float>(784, 512, "relu"),
    vc::nn::Dense<float>(512, 256, "relu"),
    vc::nn::Dense<float>(256, 10, "none")
});

// Move to GPU
model.to("cuda");

// Train
model.fit(train_dataset, 10, 0.01, 128);

// Save model
model.save("mnist_model.json");
