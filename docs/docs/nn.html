Neural Network Layers

vc::nn::Dense<T> — Fully Connected Layer

Forward: Y = (X × W) + B, optional ReLU

Constructor

Dense(size_t inputs, size_t neurons, std::string activation = "none")
  • Xavier/Glorot uniform initialization: limit = sqrt(6 / (in + out))
  • Bias initialized to zero
  • Both weight and bias have requires_grad = true

Methods

  • Tensor<T> operator()(const Tensor<T>& x) — forward pass
  • vc::vector<Tensor<T>*> parameters() — returns [&weight, &bias]
  • Dense<T> to(const std::string& device) — device transfer

Loss Functions

vc::nn::MSELoss<T> — Mean Squared Error

  • Tensor<T> operator()(const Tensor<T>& predictions, const Tensor<T>& targets)
  • Formula: loss = mean((pred - target)²)
  • Gradient: 2(pred - target)

vc::nn::CrossEntropyLoss<T> — Cross Entropy with Softmax

  • Tensor<T> operator()(const Tensor<T>& logits, const Tensor<T>& targets)
  • Numerically stable softmax + negative log likelihood
  • Gradient: (softmax(logits) - targets) / batch_size
Warning: For CrossEntropyLoss, the targets tensor must be one-hot encoded.

Example: Complete Workflow

// 1. Create layers and data
vc::nn::Dense<float> layer1(784, 128, "relu");
vc::nn::Dense<float> layer2(128, 10, "none");
vc::nn::CrossEntropyLoss<float> criterion;
vc::optim::SGD<float> optimizer({layer1.parameters(), layer2.parameters()}, 0.01);

// 2. Forward pass
auto hidden = layer1(inputs);
auto logits = layer2(hidden);

// 3. Compute loss
auto loss = criterion(logits, targets);

// 4. Backward pass
loss.backward();

// 5. Optimizer step
optimizer.step();
optimizer.zero_grad();
