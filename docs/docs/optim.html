Optimizers

vc::optim::SGD<T> — Stochastic Gradient Descent

Constructor

SGD(vc::vector<Tensor<T>*> params, float learning_rate)

Methods

  • void zero_grad() — resets all gradients to zero
  • void step() — applies W = W - lr * grad (uses cuda_sgd_kernel on GPU)
Tip: Always call zero_grad() before backward(). Gradients accumulate by default — this is intentional for gradient accumulation techniques, but forgetting to zero them is the #1 training bug.

The Standard Training Loop Pattern

for (int epoch = 0; epoch < epochs; epoch++) {
    optimizer.zero_grad();   // 1. Clear gradients
    auto output = model(x);  // 2. Forward pass
    auto loss = criterion(output, target); // 3. Compute loss
    loss.backward();         // 4. Backward pass
    optimizer.step();        // 5. Update weights
}
